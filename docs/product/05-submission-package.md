## 1. 게임 제목

`바닥이 사라지는 술래잡기`

## 2. 게임 설명

무너지는 타일 아레나에서 상대를 밀어내고 끝까지 살아남는 짧은 브라우저 파티 액션
게임이다. 플레이어 한 명과 규칙 기반 컴퓨터들이 같은 물리 규칙으로 싸운다. WASD,
방향키, 마우스 드래그, 터치 조이스틱, 게임패드로 이동하고 키보드·터치·게임패드로
밀치기와 회피를 한다.

가벼우면 빠르게 움직이고 잘 피하지만 멀리 날아가고, 무거우면 둔하지만 밀림에 강하다.
상대의 밀치기를 옆으로 피하면 공격자가 남은 관성 때문에 휘청거리며 스스로 떨어질 수
있다. Iron Boots, Feather, Spring Glove 아이템은 위험한 가장자리 쪽에 더 자주 나타나서,
먹으러 갈지 포기할지 선택하게 만든다.

8·16·24·32명 프리셋과 Easy·Normal·Hard 컴퓨터 난이도, Slow·Normal·Fast 붕괴 속도를
지원한다. 32명은 정교한 경쟁 모드가 아니라 일부러 빽빽한 `Mayhem` 모드다. 전투 결과는
숨겨진 확률이 아니라 고정 틱, 위치, 속도, 몸무게, 행동 타이밍으로 결정된다.

## 6. 제작 과정

처음에는 단순히 사람이 많은 밀치기 게임을 만들 생각이었지만, 숫자만 늘리면 승패가
읽히지 않는다는 문제가 생겼다. 그래서 렌더러와 무관한 60 Hz 고정 틱 시뮬레이션부터
만들고 이동, 약한 접촉, 밀치기 준비·판정·회복, 회피, 헛밀치기 관성, 동시 충돌, 낙하를
각각 재현 가능한 규칙으로 분리했다.

그다음 컴퓨터를 순수 랜덤으로 움직이지 않고, 공개된 화면 정보만 늦게 보고 행동 후보에
점수를 매기는 유틸리티 방식으로 만들었다. Easy·Normal·Hard는 물리 보너스가 아니라 반응
지연과 판단 간격만 바꾼다. 참가자는 8·16·24·32명으로 고정하고, 인원별 아레나와 아이템
상한을 함께 조절했다.

플레이 시간이 너무 짧아지는 문제에는 참가자를 더 늘리는 대신 붕괴 속도 선택을
추가했다. 같은 시드로 Slow·Normal·Fast를 비교하는 감사에서 평균 시간이 순서대로
길어지는지 확인했다. 몸무게와 아이템도 승률이 한쪽으로 극단적으로 쏠리지 않는지
통제된 고정 시드 실험으로 검사했다. 아이템은 바깥 두 타일 띠에 80% 이상 나타나도록
만들어, 보상을 얻으려면 낙하 위험을 감수하게 했다.

마지막으로 결과 화면에서 시드, 설정, 완료 틱, 승자, 상태 해시를 개인정보 없이 JSON으로
복사할 수 있게 했다. 자동 통계는 재미를 증명하지 못하므로, 실제 사람의 이동 이해,
회피 이해, 죽음 설명, 재시작, 가장자리 아이템 선택을 별도 플레이테스트 절차로 남겼다.
